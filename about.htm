The dispatcher

Every request fires a WEB TCP INTERRUPT. The handler reads the first line of the HTTP request, scrapes the URI, picks an HTML file by name, and hands the (pcb, filename) pair to WEB PAGE. PicoMite then streams the file out, evaluating {...} expressions inline against current BASIC state.

Pages are static .htm files on B:; only the routing and the per-page state-prep run as code. Adding a new page is "drop a file, add a route line."
